- **الميزات**: 
  - دعم البحث باللغة العربية.
  - نتائج منظمة مع عناوين بارزة.
##### 3️⃣ **حالة الطقس** 🌤️
- **الواجهة**: منطقة عرض معلومات + مؤشر تحميل.
- **العملية**: 
  1. تحديد الموقع الجغرافي تلقائياً.

- **الميزات**: 
  - عرض درجة الحرارة بمئوية.
  - رسائل واضحة عند رفض الوصول للموقع.
##### 4️⃣ **مولد رموز QR** 📱

- **العملية**: 
  - توليد رمز QR فوري عند الكتابة.
  - دعم أي طول نصي.
- **الميزات**: 
  - أعلى مستوى تصحيح للقراءة السهلة.
  - حفظ الرمز كصورة مباشرة.
##### 5️⃣ **التقويم الهجري والميلادي** 📅
- **الواجهة**: منطقتان لعرض التاريخين مع فاصل.
- **العملية**: 
  - الحصول على التاريخ الحالي.
  - تحويله للهجري والميلادي.
  - تنسيقه باللغة العربية.
- **الميزات**: 
  - عرض كامل (يوم، أسبوع، شهر، سنة).
  - استخدام الأرقام العربية.
##### 6️⃣ **الدردشة مع الذكاء الاصطناعي** 🧠
- **الواجهة**: 
  - قائمة منسدلة لاختيار النموذج.
  - نافذة دردشة + حقل إدخال + زر إرسال.
  - مؤشر تحميل + منطقة أخطاء.
- **العملية**: 
  1. اختيار النموذج المناسب.
  2. تحسين الرسالة تلقائياً.
  3. إرسال الطلب لواجهة برمجة التطبيقات.
  4. عرض الرد مع طابع زمني.
- **الميزات**: 
  - محادثات مستمرة مع حفظ السياق.
  - فصل واضح بين رسائل المستخدم والذكاء الاصطناعي.
##### 7️⃣ **تفسير الأحلام** 🌙

- **العملية**:
